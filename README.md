# 📖 About

**WeCare Hospitals** is a modern, responsive healthcare website designed with a clean, professional interface focused on delivering an excellent patient experience.

The project includes appointment booking, doctor information, hospital departments, and a polished landing page suitable for hospitals, clinics, and healthcare organizations. The live version is deployed on Netlify. :contentReference[oaicite:0]{index=0}

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/uzi-makes/wecare-hospitals.git
```

Go inside the folder

```bash
cd wecare-hospitals
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
